## X / Telegram 课程判断规则

购买前检查：

- 讲师有公开高质量报告、研究文章或平台履历。
- 课程最近 12 个月更新。
- 有真实 lab、作业、答疑或报告批改。
- 不只是工具命令合集。
- 从讲师官网、官方 Gumroad/Teachable/Udemy/Discord 购买。
- Telegram/fcmit.cc 只作为“目录发现源”，不要默认信任盗版/过期包。
